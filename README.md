# Legal Tools Suite

A collection of web-based tools for legal case management.

## Projects

### [Lighthouse](./lighthouse/) - Case Management
Track cases, documents, court dates, and trial preparation. A comprehensive dashboard for managing your legal workload.

### [Bell Tower](./bell-tower/) - Case Scheduling
Assign the right cases to the right court days. Optimize scheduling and manage courtroom capacity with conflict detection.

## Getting Started

Open `index.html` in a modern web browser to access the landing page, which provides navigation to both projects.

Or access each project directly:
- Lighthouse: `lighthouse/index.html`
- Bell Tower: `bell-tower/index.html`

## Deployment

Hosted on GitHub Pages: https://rramkhel.github.io/lighthouse/

## Tech Stack

Both projects use the same technology stack:
- Vanilla JavaScript (no frameworks)
- Tailwind CSS (CDN)
- Lucide Icons (CDN)
- Hash-based routing
- No build step required
